# Analisis Sentimen IMDb

## Deskripsi

Proyek ini melakukan analisis sentimen ulasan film menggunakan metode Bag of Words dan TF-IDF dengan model Naive Bayes.

## Dataset

IMDb Movie Review Dataset (50.000 data)
FIle dataset (https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews)

## Cara Menjalankan

1. Upload dataset IMDB Dataset.csv
2. Jalankan notebook dari atas (Run All)
3. Lihat hasil evaluasi dan visualisasi

## Hasil

* BoW Accuracy: 84.56%
* TF-IDF Accuracy: 85.20%
